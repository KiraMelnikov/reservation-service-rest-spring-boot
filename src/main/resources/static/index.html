Reservation Service API

Try the REST methods exposed by ReservationController · Customer site → · Admin dashboard →

Base URL

GET /health

Service health check.

GET /reservations

Returns all reservations.

GET /reservations/{id}

Returns reservation by UUID.

POST /reservations

Creates a reservation. All fields are required.

PUT /reservations/{uuid}

Updates a reservation. Only PENDING reservations can be modified.

POST /reservations/{uuid}/approve

Approves a reservation. Fails if there is a date/room conflict with another non-approved reservation.

DELETE /reservations/{uuid}

Cancels a reservation (sets status to CANCELLED).
